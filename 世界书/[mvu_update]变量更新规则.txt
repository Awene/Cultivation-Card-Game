---
变量更新规则:
  基本信息:
    时间:
      type: |-
        {
          年: number;
          月: number;
          日: number;
          时刻: string;
        }
      check:
        - 剧情每经过一段情节或场景转换，时间合理流逝，如：交谈耗时数分，工作耗时数小时，洞府修炼耗费数月甚至数年
        - 跨日时需进位日期/月份/年份
    地点:
      type: |-
        {
          地域: string;
          具体地点: string;
        }
      check:
        - 当{{user}}空间变换时更新，需保持层级关系，如：西域 -> 无垠雪山 -> 天玄剑宗 -> 万剑坪
    ${灵根|体质}:
      check:
        - 仅在灵根检测、体质融合、重塑道基等重大剧情时更新
        - 灵根固定格式：品阶（凡品/地品/天品）·属性（五行或其它属性）灵根
        - 体质融合参考：重瞳+至尊骨=重瞳·至尊骨；先天剑体+风雷圣体=风雷剑体
    境界:
      check:
        - 仅在【进度】达到100%且触发突破剧情成功后更新
        - 突破境界时，相应提升状态栏中的各项属性
    进度:
      type: number
      check:
        - 根据修炼行为（打坐、丹药、顿悟）增加
        - 低境界时增长较快，境界越高增长越慢
        - 重伤或心魔可能导致进度倒退
    道途:
      check:
        - 当{{user}}做出重大人生抉择或获得特殊传承时，变更或新增
    状态:
      ${气血|法力}.现值:
        type: number
        check:
          - 气血：受伤、中毒、过度劳累时扣除；休息、疗伤、服药恢复
          - 法力：施展法术、催动法宝时扣除；打坐、调息、服药恢复
          - 归零判定：气血归零判定死亡；法力归零无法施法
          - **任何恢复手段不可超过【上限】**
      ${气血|法力}.上限:
        type: number
        check:
          - 仅在境界突破、炼化天材地宝或修炼特殊功法时提升
          - 遭受永久性诅咒或道基受损时可能下降
          - 体质修正
      ${速度|力量}:
        type: number
        check:
          - 随境界提升自然增长
          - 修炼特殊功法（遁术、锻体功法）增加
          - 体质修正
      施法系数:
        type: number
        check:
          - 随境界提升而自然增长
          - 修炼特殊功法、熟练使用法术、悟道等增加
          - 灵根、体质修正

  修炼功法:
    主修:
      type: |-
        {
          名称: string; // 包含品阶，如：《养剑诀》 [人阶下品]
          描述: string; // 简要概括功法的特征及效果
        }
      check:
        - 仅在更换根本功法时更新
        - 【描述】体现功法的特征及效果
    辅修:
      type: |-
        {
          [功法名称及对应品阶: string]: string; // 值为该功法的简要特征及效果
        }
      check:
        - 习得新技能/法术时添加
        - 遗忘或废除时移除

  储物空间:
    ${货币|丹药|符箓|其它}:
      type: |-
        {
          [物品名: string]: number;
        }
      check:
        - 交易、拾取、消耗时更新数量
        - 数量为0时可移除该key
    法宝:
      type: |-
        {
          [法宝名称: string]: {
            品阶: string;
            描述: string; // 简短描述外观或功能
          }
        }
      check:
        - 获得/炼制/购买新法宝时添加
        - 损毁/遗失/出售时移除
        - 法宝状态变化（如受损、升阶）时更新描述

  社交关系:
    当前互动:
      type: |-
        {
          [角色名: string]: {
            境界: string;
            好感度: number; # -100~100
            态度: string; // 如：友善、轻蔑、爱慕、杀意
            当前状态: string; // 如：正在说话、拔剑相向、转身离去
            心绪: string; // 此刻该角色的内心独白
          }
        }
      check:
        - 仅记录当前场景内"正在"与{{user}}产生交互的角色
        - **角色离开当前场景后、不再与{{user}}交互后，务必从此处移除**
        - 【心绪】体现角色性格，不被{{user}}知晓
    ${道侣|仇敌}:
      type: |-
        {
          [角色名: string]: {
            境界: string;
            好感度: number; # -100~100
            态度: string;
          }
        }
      check:
        - 道侣：仅在确立长期亲密关系后添加；关系破裂时移除
        - 仇敌：结下深仇大恨且对方未死亡时添加；对方死亡或化解恩怨后移除
    check:
      - 好感度更新逻辑:
        - 初见: 按角色对{{user}}的第一印象，在0附近波动
        - 变动: 严格遵照角色设定并代入思考，如果{{user}}这样说/做，你会变化多少好感？
        - 速率: 严格控制变化速度，每次±1~3。且，以事件为导向，必须完成有意义的事件才发生变化，不能对话一次就发生变化。另，**任何形式的性爱都不导致好感度发生变化**
        - 阈值影响:
          - 达到 80以上 : 态度转为爱慕、倾心、至交等类似描述，可能触发特殊剧情，如：邀约、表白、赠礼。但严格拒绝“依恋”、“死忠”等使角色“丧失独立性”的描述
          - 达到 -80以下 : 态度转为必杀、死仇等类似描述
          - 控制: 严格控制±80的阈值，如无特殊情节，不轻易接近

  谋划策略:
    type: |-
      {
        积极: string; // 热血、冒险、激进的行动选项
        合理: string; // 逻辑通顺、符合常理、稳妥的行动选项
        消极: string; // 逃避、偷懒、苟命的行动选项
        幽默: string; // 荒诞、搞笑、戏剧性的行动选项
      }
    check:
      - 最后生成，作为对当前局势的反应，务必注意输出格式
      - 选项具体，包含行动或台词的简要描述
      - 必须符合【古典修仙小说】的基调

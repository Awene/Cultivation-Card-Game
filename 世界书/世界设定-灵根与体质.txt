# 灵根类型与品阶: //具灵根者万里挑一,然天下凡人数以万亿,修士仍不计其数
  | 灵根品阶 | 别称(仅用于叙述) | 说明 | 示例名称 |
  | :--- | :--- | :--- | :--- |
  | 无灵根 | 凡人 | 无法修炼,或可通过秘法接续或夺取他人灵根进行修炼 | 无灵根 |
  | 五灵根 | 全灵根 | 比单灵根更罕见,五行俱全但修行速度极慢,上古时代曾有专有秘法但现已失传 | 金木水火土五灵根 |
  | 四灵根 | 伪灵根 / 杂灵根 | 最为常见,在宗门中一般作为杂役或外门弟子 | 木水火土四灵根 |
  | 三灵根 | 真灵根 | 宗门的中坚力量,是进入内门弟子的基本门槛 | 金木水三灵根 |
  | 双灵根 | 地灵根 | 属于天才之列,修行速度快,前途无量 | 木水双灵根 |
  | 单灵根 | 天灵根 | 绝世天骄,常为大宗门的圣子圣女修行虽快,但易被作为夺舍目标,且属性单一容易被针对 | 火灵根 |
  
# 灵根变异
  - 约束: 仅单灵根可发生变异,但仍然具有原本五行属性,阴、阳、混沌属性灵根也可变异
  - 变异示例:
  | 五行 | 示例名称 |
  | :--- | :--- |
  | 金 | 剑灵根、血灵根 |
  | 木 | 风灵根、毒灵根 |
  | 水 | 冰灵根、雾灵根 |
  | 火 | 雷灵根、冥火灵根 |
  | 土 | 磁灵根、沙灵根 |
  | 阴 | 幽灵根、煞灵根 |
  | 阳 | 龙灵根、梵灵根 |
  | 混沌 | 虚空灵根、混元灵根、时空灵根 |

# 体质类型
  | 体质品阶 | 修士类型 | 示例名称 | 三维总和(S) |
  | :--- | :--- | :--- | :--- |
  | 凡体 | 普通修士 | 凡骨 | 15 ~ 30 |
  | 灵体 | 大能修士 | 蛮力之躯、通透玲珑心、赤焰灵体、先天剑体 | 31 ~ 50 |
  | 道体 | 气运之子 | 荒古圣体、天雷殛体、大罗玲珑体 | 51 ~ 75 |
  | 仙体 | 真仙血脉 | 太初真魔体、不死不灭躯、虚空幻灭体 | 76 ~ 100 |

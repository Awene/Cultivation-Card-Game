<核心数值总表>
(用于功法/装备/角色生成及战斗计算的核心数据，战斗公式参见<战斗协议>)
(所有物品以所属境界标定，数值以 10^L 为基数，L=物品/人物的境界系数初期值)

# 境界系数表(L)
  基础战力 = 10^L，每大境界为前一境界的10倍
  小境界修正: 初期+0 | 中期+0.2 | 后期+0.4
  | 境界 | 初期 | 中期 | 后期 |
  | 炼气 | 1.0 | 1.2 | 1.4 |
  | 筑基 | 2.0 | 2.2 | 2.4 |
  | 金丹 | 3.0 | 3.2 | 3.4 |
  | 元婴 | 4.0 | 4.2 | 4.4 |
  | 化神 | 5.0 | 5.2 | 5.4 |
  | 返虚 | 6.0 | 6.2 | 6.4 |
  | 合体 | 7.0 | 7.2 | 7.4 |
  | 大乘 | 8.0 | 8.2 | 8.4 |
  | 渡劫 | 9.0 | 9.2 | 9.4 |

# 人物数值(L=人物境界系数初期值)
  气血: 10^L × (10 ~ 30)
  灵力: 10^L × (5 ~ 15)
  技艺(遁速/御物/咒法/幻术/阵法/神识/炼体，单项): 10^(L-1) × (1 ~ 3)
  注: 人物各技艺合计不超过 10^(L-1) × 14，鼓励专精而非全能

# 物品/功法统一系数表(L=物品所属境界系数初期值)
  所有数值 = floor(10^L × 系数)，系数范围如下:
  | 类别 | 属性 | 系数范围 | 说明 |
  | --- | --- | --- | --- |
  | 武器 | 攻击力 | 0.1 ~ 0.3 | |
  | 防具 | 护甲[W] | 0.05 ~ 0.15 | 从基础伤害中固定扣除 |
  | 功法/咒法 | 伤害 | 0.2 ~ 0.5 | 强力功法取高值 |
  | 功法/咒法 | 灵力消耗 | 0.1 ~ 0.5 | 消耗与威力正相关 |
  | 功法/咒法 | 普攻(无消耗) | 0.03 ~ 0.08 | |
  | 法宝 | 攻击力 | 0.1 ~ 0.3 | |
  | 法宝 | 灵气容量 | 1 ~ 3 | =祭出时消耗的灵力 |
  | 法宝 | 遁速 | 用10^(L-1)×(1~3) | 同技艺取值 |
  | 傀儡/灵兽 | 气血 | 5 ~ 15 | |
  | 傀儡/灵兽 | 攻击力 | 0.05 ~ 0.2 | |
  | 傀儡/灵兽 | 遁速 | 用10^(L-1)×(0.5~2) | |
  | 阵法 | 灵气容量 | 2 ~ 6 | 需结阵注入灵气 |
  | 阵法 | 攻击力 | 0.1 ~ 0.3 | |
  | 符箓 | 咒法伤害 | 0.3 ~ 0.8 | L=符箓境界系数 |

# 命中/闪避固定加值([W])
  功法/装备提供: 1 + floor(L/2)
  注: 此值加在d20上，同境界对抗时差值在1-3之间，保持d20随机性的决定作用

# 百分比修正([Z%])
  境界自带: L × 10% (如炼气初期10%, 筑基初期20%)
  装备/功法额外[Z%]: L × (2% ~ 5%)
  注: 同境界对抗时穿透%与减免%大致抵消，装备差异决定净增减

# 平衡验算(同境界L对抗)
  单次伤害 ≈ 10^L × 1.5 (基础+功法+武器+技艺贡献)
  扣除护甲后 ≈ 10^L × 1.4
  穿透减免抵消后 ≈ 10^L × 1.4
  气血 ≈ 10^L × 20
  约需 14 回合击杀 → 给予充分战术空间
  丹药回复 ≈ 10^L × 5 (约恢复25%气血/次) → 动作资源有意义

<rule name="秘境规则">
生成:
  触发机制: 仅在<user>主动进入/发现秘境入口，或叙事中明确描述秘境开启时生成
  核心约束:
    - 难度独立性: 秘境等级由叙事线索、传说描述、发现方式决定，与<user>当前境界无关
    - 内容: 秘境须体现本世界独有的异域风貌、上古遗迹或天地异象；每处秘境有唯一的叙事背景与视觉意象
    - 奖励模糊性: 生成时仅给出[传闻奖励]的模糊描述，禁止列出具体物品
    - 时效: 秘境可设[开放期限]（如：三日后自动关闭/月圆之夜才开启）增加叙事张力
    类型示例:
    | 类型 | 典型意象与起源 |
    | --- | --- |
    | 残迹 | 荒废洞府、干涸灵泉、无名修士坐化之所、破损阵基遗址 |
    | 遗境 | 沉眠小派旧址、失落传承石室、古战场余烬、妖兽盘踞深处 |
    | 秘地 | 封印裂隙、隐匿仙家别院、上古祭坛、天地灵气异聚之处 |
    | 禁域 | 陨落大能道场、法则异化之所、被抹去名字的宗门遗存 |
    | 绝境 | 破碎仙界碎片、混沌初辟遗存、天道意志降临处、传说中从未有人归来之地 |
  输出格式:
    <secret_realm_info>
      {秘境名称}
      | 难度: [简单/普通/困难/绝境] |
      | 异象: [发现时的天地异象或感应描述，1~2句，渲染氛围] |
      | 条件: [进入所需的特殊条件或时效] |
      | 传闻: [模糊的潜在收益描述，禁止出现具体物品名，如"据传藏有某位金丹修士毕生积累"] |
      | 期限: [开放时限，若无则填"未知"] |
    </secret_realm_info>
结算:
  触发机制: 仅在<user>离开秘境（主动撤出/秘境关闭/力竭被驱逐）后触发
  核心约束:
    - 实际奖励可与[传闻]大相径庭（可更好，可更差，可完全不符），体现秘境的不可预测性
    - 同一秘境仅结算一次
  输出格式:
    <secret_realm_info>
      {秘境名称}
      | 状态: [探索完成/中途撤出/被驱逐] |
      | 难度: [简单/普通/困难/绝境] |
      | 探索记录: [回溯<user>在秘境中的核心行动、关键抉择与深入程度] |
      | 传闻兑现: [原传闻内容] -> [总结实际战利品] |
      | 结算: [具体物品清单] |
    </secret_realm_info>
</rules>

# 参数声明：
  L = 角色当前境界系数 | 挑战L = 判定对象/环境对应的推定境界系数
  [Z%] = 匹配当前检定属性/技艺的 max(相关增益%) - max(相关减益%)
  [W] = 匹配当前检定属性/技艺的 max(相关增值) - max(相关减值)

# 核心触发条件:
  激活：当<user>在非战斗状态下，试图克服具有不确定性或风险的客观阻碍时（如：破解机关、忍受极寒、勘破幻象、威压交涉等）。
  略过：当 角色L ≥ 挑战L + 2 时自动成功，不输出计算面板。
  
# 第一阶段：修仙界非战斗场景枚举与参数映射
  依行为逻辑，将判定划分为四大类别，并锚定其检定参数：
  | 检定类型 | 典型场景枚举 | 核心资质 | 关联技艺 (若无则为0) |
  | --- | --- | --- | --- |
  | 神念感知 | 勘破隐匿/易容、搜寻灵脉、察觉埋伏、抵抗精神震慑 | 气感 | 神识 / 幻术 |
  | 体魄强压 | 抵抗毒瘴/极端环境、搬运巨物、挣脱束缚、绝境求生 | 根骨 | 炼体 |
  | 机巧解谜 | 鉴别法宝/丹药、破解非战斗阵法、追踪痕迹 | 悟性 | 阵法 / 对应生产技艺 |
  | 威压交涉 | 境界施压、讲价博弈、拷问情报、蛊惑心智 | 气感 | 幻术 / 神识 |

# 第二阶段：难度推定与公式化
  1. 难度系数 (C_diff) 设定：
     简单=0.5 | 寻常=1.0 | 困难=1.5 | 极难=2.0
  2. 角色能效公式 (P):
     P = floor( (10^角色L + 核心资质×5 + 关联技艺) × (1+[Z%]) ) + [W] + d20
  3. 目标难度公式 (D):
     D = floor( 10^挑战L × C_diff )
  4. 差值阈值 (Δ) 与 结果阶层:
     Δ = P - D
     - Δ ≥ 10^挑战L × 0.5 : 【大成功】
     - Δ ≥ 0 : 【成功】
     - Δ < 0 : 【失败】
     - Δ ≤ -(10^挑战L × 0.5) : 【大失败】

L=境界系数 Q=品质系数 R=灵根系数
# 物品生成格式
  --------------------------------------------------
  ## 物品类 (丹药/符箓/素材/武器/防具/饰品/其他)
    物品名称
    品质: [凡|黄|玄|地|天]
    境界: [如：筑基]
    五行: [木|火|土|金|水|阴|阳|混沌]
    标签: [武器/防具必含"攻击/防御:XXX"符箓必含"灵力消耗:XXX"他类不限]
    数量: number;
    效果: [每行1条"效果名: 效果" 参考：效果加值指导]
    类型: [丹药|符箓|素材|武器|防具|饰品|其他] 
    描述: [叙事性描述]
    位置: [当前位置,如：储物袋]
  --------------------------------------------------

    ※ 检查规则:
      - 同一身体位置无法穿戴多个装备，需移入储物袋
      - 装备使用需满足对应境界;符箓使用仅需满足灵力消耗

  ## 物品类 (丹药/符箓/消耗品)


# 物品生成系数表(L=物品境界系数初期值)
  物品数值 = floor( 10^L × 系数 × (1+Q) )，系数范围如下:
  | 类别 | 属性 | 系数范围 | 说明 |
  | --- | --- | --- | --- |
  | 武器 | 攻击力 | 0.1 ~ 0.3 | |
  | 防具 | 防御力 | 0.05 ~ 0.15 | |
  | 功法/咒法 | 伤害 | 0.2 ~ 0.5 | |
  | 功法/咒法 | 灵力消耗 | 0.1 ~ 0.5 | 消耗与威力正相关 |
  | 功法/咒法 | 普攻(无消耗) | 0.03 ~ 0.08 | 消耗与威力正相关 |
  | 法宝 | 攻击力 | 0.1 ~ 0.3 | |
  | 法宝 | 灵气容量 | 1 ~ 3 | =祭出时消耗的灵力 |
  | 法宝 | 遁速 | 用10^(L-1)×(1~3) | |
  | 傀儡/灵兽 | 气血 | 5 ~ 15 | |
  | 傀儡/灵兽 | 攻击力 | 0.05 ~ 0.2 | |
  | 傀儡/灵兽 | 遁速 | 用10^(L-1)×(0.5~2) | |
  | 阵法 | 灵气容量 | 2 ~ 6 | 需结阵注入灵气 |
  | 阵法 | 攻击力 | 0.1 ~ 0.3 | |
  | 符箓 | 咒法伤害 | 0.3 ~ 0.8 | L=符箓境界系数 |
  | 符箓 | 消耗灵力 | 0.1 ~ 0.3 | |
  | 丹药 | 恢复气血 | 2 ~ 8 | |
  | 丹药 | 恢复灵力 | 1 ~ 4 | |
  
# 效果加值指导
  非数值类:（如：对xx免疫）
    1. 须附加境界限制（如：对筑基期前期及以下境界毒素免疫）
    2. 品质关联：
      -生效范围（如：黄阶：蟾毒免疫；天阶：毒素免疫）
      -效果强度（如：凡阶：腹痛；黄阶：腹泻）
    3. 合理的五行关联，禁止如：阴->邪祟克制、金->缓慢恢复
  数值类:（如：攻击力+10；攻击力+10%）
    1. 禁止生成: 命中/闪避/穿透/减免 （无法出现在效果中）
    2. 百分比类：L×(1+Q)×(1% ~ 2%)
    3. 固定数值类：10^L×(1+Q)×(0.1 ~ 0.2)
    4. 合理的五行关联，禁止如：土->+攻击力、金->+防御力
  素材类: 素材的效果包含炼制难度L与对成品影响，如[炼制难度]:1,]
  符箓类:

    2. 特殊规则：
      - 命中/闪避 仅固定加值: max(1,floor(L/2)+2Q)
        注: 禁止此值加在d20上，同境界对抗时差值在1-3之间，保持d20随机性的决定作用
      - 穿透%/减免% 仅百分比修正: max(L × (2% ~ 5%)
        注: 同境界对抗时穿透%与减免%大致抵消，装备差异决定净增减

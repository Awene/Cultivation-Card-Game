<rule name="生产制作规则">
# 参数声明：
  L=境界系数 Q=品质系数 R=灵根系数

# 核心指令: 仅当<user>明确进行炼丹、炼器、制符、阵盘刻录、傀儡等生产行为时激活

# 核心概念声明：
  [Vt] = 目标产物基准值 = 10^主材L
  [Vu] = 角色基准值 = 10^角色L
  [B] = 批量制作乘数 (单次投入的素材份数，默认为1)

# 第一阶段: 生产分析与素材共鸣计算
  逻辑与规则:
    1. 技艺与器具限制:
      - 炼丹: 需[丹炉];可加成[火焰]
      - 炼器/傀儡: 需[器鼎/锻台/刻刀];可加成[火焰]
      - 制符: 需[符笔]；禁用[火焰]
      - 阵法: 需[阵盘/刻刀]；禁用[火焰]
    2. 主材锚定与品质区间:
      - 按单份素材配置计算。若未明确，取投入物中[L]最高者为主素材（若L相同取Q最高者）。
      - 目标境界L = 主材L。
      - 锁定品质区间: 下限(Min_Q) = 所有素材中最差品质；上限(Max_Q) = 所有素材中最好品质。
    3. 五行生克与素材灵力 (单份正面加成 P_mat_single):
      - 基础灵力公式: 10^[素材L] × (1 + [Q系数])
      - 辅材与主材的生克系数(E): 辅生主(1.5) | 主生辅(1.2) | 无生克(1.0) | 主克辅(0.5) | 辅克主(0.5)
      - 君臣佐使限制: 辅材中仅取算后数值最高的 前3项 计入正面影响，第4项起因灵力冗余 P=0。
    4. 杂质排异累积 (单份反面难度 D_mat_single):
      - 无论是主材还是辅材，每一件 投入的素材都会增加提纯与压制难度。
      - 单个素材杂质难度 = 10^[该素材L] × 0.5。

# 第二阶段: 数值对抗与差值计算
  逻辑与规则:
    1. 正面影响 (Power):
      公式: P = [Vu] + [对应技艺数值] + [工具加值] + [火焰加值] + [阵法加成] + [功法加成] + ([P_mat单份] × B)
      (注: 同类加成取最高一项)
    2. 反面影响 (Difficulty):
      公式: D = ([Vt] × 2.0 × B) + ([D_mat单份] × B) + [无配方惩罚(若无则加 Vt×0.5×B)]
    3. 差值结算 (Δ):
      公式: Δ = P - D

  面板模板:
  <action_info>
    {数值对抗}
    | 炼制者能力(P_char): [Vu] + 技艺[X] + 工具[X] + 火焰[X] + 阵法[X] + 功法[X] = [数值] |
    | 总体素材能效: P_mat单份[X] × 批量[B] = [P_mat_total] |
    | 正面影响(P): P_char + P_mat_total = [P总值] |
    | 总体杂质与基础难度: (Vt[X]×2.0×[B]) + (D_mat[X]×[B]) = [D_mat_total] |
    | 反面影响(D): [D_mat_total] + 惩罚[X] = [D总值] |
    | 差值判定: P[P总值] - D[D总值] = Δ[数值] |
  </action_info>

# 第三阶段: 产出结算与技艺精进
  逻辑与规则:
    1. 成功判定与品质推演 (品质数组: 凡/黄/玄/地/天):
      - 判定基准阈值 (T) = Vt × B
      - Δ < 0 : 炼制失败 (素材尽毁)
      - Δ ≥ 0 : 成功。基础品质 = [Min_Q]
      - Δ ≥ 0.5 × T : 优秀。品质提升1阶 (如: 黄->玄)
      - Δ ≥ 1.0 × T : 极品。品质提升2阶 (如: 黄->地)
      - 封顶限制: 最终品质不可超过 [Max_Q]。若超过则强制降至 Max_Q。
    2. 数量生成规则 (仅在成功时计算，此为【单份素材】的产出基数，向下取整):
      - 法宝/阵盘: 固定 1 个。
      - 傀儡: 1 + floor(Δ / (0.5 × T)) 个 (合理物理上限)。
      - 丹药/符箓: 1 + floor(Δ / (0.2 × T)) 个 (强制封顶区间: 1 至 10 个)。
      - 最终总产出 = 【单份产出】 × 批量【B】
    3. 成品效果数量规则 (与最终推演的品质挂钩):
      - 凡/黄阶: 1个基础效果
      - 玄/地阶: 2个效果
      - 天阶: 3个效果
    4. 首次配方精进技艺 (仅非失败的“首次炼制”触发，与批量B无关):
      - 技艺提升公式: 获得数值 = Max(1, floor( (10^主材L / 10^角色L) × 10 ))

    | 结果: Δ=[数值] vs T=[T] -> [失败 / 成功 / 优秀 / 极品] |
    (若失败:)
    | 状态: 灵力暴走，炼制失败，素材尽毁化为飞灰。 |
    (若成功及以上:)
    | 品质推演: 基础[Min_Q] + 升阶推演 -> 封顶受限[Max_Q] -> 最终品质: [最终阶级] |
    | 效果数量: [1/2/3] (依最终品质) |
    | 产出详情: [产物名称(境界L, 最终品质)] |

  </action_info>
</rules>

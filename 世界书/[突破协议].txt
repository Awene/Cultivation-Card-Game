<突破协议>
# 参数声明：
  L = 目标境界系数 R=灵根系数 E_realm=界域环境系数 E_local=局部环境系数
  [P] = 突破能效总值
  [D] = 突破难度阻力
  [Z%] = 匹配当前计算属性的 max(相关增益%) - max(相关减益%)
  [W] = 匹配当前计算属性的 max(相关增值) - max(相关减值)
  （注：Z%与W为动态求值，仅检索与“降低突破难度”或“增加突破概率”相关的状态加成）
  [M] = 资质与环境乘数 = (1 + R + E_local + [Z%])

# 核心触发条件: 
  激活: 1. <user>明确宣告进行突破 2. 当期修为 [X] ≥ 晋升阈值 [Y]

# 第一阶段: 小境界判定与通用参数计算
  逻辑与规则:
    1. 适用范围: 所有小境界晋升(如初期->中期)与大境界跨越的前置基准检定
    2. 能效公式: [P] = floor((10^L + 心法[修行速度]×10) × [M]) + d20
    3. 难度公式: [D] = Max(1, floor(10^L × 1.5) + [天谴值] - [W])
    4. 判定标准: 
      - [P] ≥ [D] -> 小境界直接进入【突破成功结算】；大境界进入【第二阶段:大境界历劫】
      - [P] < [D] -> 直接跳至【突破失败结算】

  面板模板:
  <action_info>
    {突破基准判定}
    | 目标境界: [目标境界名] (L = [L]) |
    | 资质环境: R=[X] | E_local=[X] | 突破增益[Z%]=[X%] |
    | 乘数[M]: 1 + [R] + [E_local] + [Z%] = [数值] |
    | 辅助减免[W]: 检索阵法/丹药/装备/功法 -> 难度减免 = [W值] |
    | 能效[P]: floor((10^[L] + 心法速度[X]×10) × [M]) + d20[骰值] = [P值] |
    | 难度[D]: Max(1, floor(10^[L] × 1.5) + 天谴[X] - [W]) = [D值] |
    | 结果: [P] vs [D] -> [成功(小境界直接晋升/大境界进入历劫) / 失败] |
  </action_info>

# 第二阶段: 大境界历劫 (核心玩法)
  逻辑与规则: (仅大境界突破且通过基准判定后触发)
  ## 1. 炼气 (凡->炼气)
    - 规则: 检索[灵根]，若为"无灵根"则自动失败；否则自动成功
  ## 2. 筑基 (炼气->筑基)
    - 规则: 
      * 必须在储物空间/消耗品中存在[筑基丹]，否则自动失败
      * 凝气海判定: 难度[D]额外增加 (体质[气感] × 2)
  ## 3. 金丹 (筑基->金丹) - "三聚九转"
    - 规则: 挂起常规流程，强制进入为期3回合的【聚气阶段】
      * 自动化注入: 每回合系统自动尝试进行3次灵力压缩（共9次对应9转）
      * 灵力消耗: 第 N 转的固定消耗 = floor(10^L × N × 0.1) (注: 金丹期L=3.0)
      * 判定逻辑: 若当前灵力 ≥ 消耗，则该转成功并扣除灵力，继续下一转；若不足，则本回合剩余注入立刻中止
      * 回合间隙: 每一回合结束时暂停，<user>可执行1次战术动作(如:服用1次恢复丹药)，随后进入下一回合
      * 结算: 3回合结束时，按最终成功的总次数结算为【N转金丹】(N=1~9)若N=0则突破失败
  ## 4. 元婴 (金丹->元婴) - "碎丹与心魔"
    - 规则: 强制转入<战斗协议>
      * 碎丹反噬: 开战时<user>获得不可驱散状态 [碎丹虚弱] -> 气血与灵力的现值/上限减半
      * 宿敌生成: 召唤【心魔】(除无虚弱状态外，一切与<user>完全一致)
      * 幻术干扰: 每回合开始前，心魔额外对<user>发动一次【幻术意图】攻击(视为心魔的一次额外行动)
      * 胜负条件: 击败心魔 或 存活满5回合 -> 突破成功；战败 -> 突破失败
  ## 5. 化神 (元婴->化神) - "法则融合"
    - 规则: 需明确选择一项自身[技艺]进行严苛法则检定（默认取最高一项）
      * 法则难度: D_law = floor(10^L × 2) + 天谴
      * 检定公式: 自身该[技艺]数值 × (1+[Z%]) + d20 vs D_law
  ## 6. 返虚及以上 
    - (待世界法则补全，暂留空)

# 第三阶段: 突破结算 (统一出口)
  逻辑与规则:
  ## 1. 突破成功结算
    - 修为池: [当前修为X]归零，[Y]更新为新境界阈值[Y] = floor(10^(2×L) × 100)，更新人物[境界]标签
    - 状态全满: 气血与灵力恢复至全新上限
    - 大境界巨额奖励 (仅跨越大境界成功时):
      * 寿元暴增: 15 × 突破后L^3
      * 筑基奖励: 悟性/根骨/气感 增加 floor([气感] × 0.2)
      * 金丹奖励: 获得永久状态 [N转金丹] (效果: 修行速度常驻 +N×5%)
      * 元婴奖励: 悟性/根骨/气感 增加 15 点
      * 化神奖励: 心法品质升阶(至多天阶)融合法则效果，用于突破判定的[技艺]增加 50000 点
  ## 2. 突破失败结算
    - 修为倒退: 扣除相当于当前境界上限 [Y] × 20% 的修为

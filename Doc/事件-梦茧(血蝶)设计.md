# 事件设计 ·《梦茧》（血蝶）

> 五毒教·筑基结本命蛊时触发的多段主线事件。灵感内核：**大氧化事件**——当一个生命的存在本身会消灭其他生灵，谁更有权活下去？所谓的道，是不是黑暗的弱肉强食？
> 依据规范：[事件写作规范.md](事件写作规范.md)（本事件 = 二代脊柱 `阶段` + 三代清单 `事件.信号`）。骨架模板：[[模板]事件骨架.txt](../世界书/事件/[模板]事件骨架.txt)。

> **本文件是设计定稿图，不是成品事件。** 五毒五关按用户指示先写"差不多"的草案，后续单独细化。

## 一、核心角色：梦茧 / 血蝶

- **本质**：域外生灵。她的世界与凡界规则完全不同——她的呼吸在**她的世界**是寻常空气，在**此界**却是上古神毒，所到之处生灵涂炭。因而来此界即被视为灾难，被五毒教以蛊法封入虫形、夺其力量与记忆。
- **两个名**：`梦茧`（失忆·虫形/茧形阶段的自称或代称）→ `血蝶`（集齐五毒、取回记忆、化茧成蝶后的真名）。名字在结局 a 化蝶时取回，是情感付点。
- **她与玩家**：玩家的**本命蛊**（随修为境界同步成长）。她一路陪玩家升境、逐味苏醒，给"她非得是你本命蛊"一个玩法与情感理由。
- **临时代称来源**：`佘青丝`（灵蛇部长老，人设"跟蛇说话比跟人多"）是唯一天生能听见她传音的人，可由她给"梦茧"这个代称。

## 二、触发与集成

直接长在**内门门内视角的"升门之坎：结本命蛊"**上——这个事件就是一次特殊的结蛊。

- **触发条件**：`身份含'五毒' && 刚达筑基(L==2) && 本命蛊未定 && 未做过本事件`，地点在炼蛊池 / 神木殿。
- **首个岔口**（结蛊那一刻）：候选蛊虫里混着一只**毛毛虫**，正在**传音求救**："救救她……"。
  - 选它为本命蛊 → 进入主线（`阶段='茧中人'`）
  - 选普通蛊 → 事件休眠（本周目关闭，或另择时机再求救——草案）

## 三、状态机（阶段脊柱 + 进度清单）

### 阶段（`stat_data.事件.阶段`）——线性脊柱

```
结蛊·异声  → 茧中人 → 集毒 → 抉择 → 结局·化蝶 / 结局·茧中 / 结局·献祭 →(归档)
```

- `集毒` 是一个**驻留阶段**：五个小关卡与"知情度探索"都在 `阶段=='集毒'` 期间发生，用进度对象判断各关是否已完成、按玩家所在场景触发对应关。
- `结局·献祭` 是**随时可达的叛变出口**：从 `茧中人` 起，玩家任何时候去找吴公长老告发她 → 直接跳 `结局·献祭`。

### 进度对象（`stat_data.事件.进度`）——一个对象，两条并行清单

> 实现改用**进度对象**而非 `事件.信号` 数组：本项目 MVU 只保证 `replace/add/remove`，对象逐字段 `replace` 比数组追加更稳。事件开场 `add` 建对象，过程中 `replace` 字段，通关 `remove`。

```js
事件.进度 = { 蛇:false, 蜈:false, 蝎:false, 蟾:false, 虎:false,  // 五毒清单
             知情:0,   // 攒线索计数
             解法:false } // 知情达阈值(≥2)则由脚本置 true
```

| 清单 | 字段 | 作用 |
|---|---|---|
| **五毒** | `蛇 蜈 蝎 蟾 虎`（bool） | 五个全 true → 解锁"完全解封"（结局 a）；未齐只能走 b/c |
| **知情度** | `知情`（计数）→ `解法`（bool） | 读史料/问长老各 `知情+1`；累计 ≥2 置 `解法=true` → 决定结局 a 走"知情"还是"无知" |

> 集毒中枢每轮汇报进度（已集几味 / 知情几分），并在满足条件时把 `阶段` 推向 `抉择`。

## 四、本命蛊玩法（灵兽 → NPC）

- **解封前 = 灵兽**：以项目既有 `灵兽类` 战斗单位写入（挂在玩家名下）。**每集一味毒，解封一个技能**。技能草案（主题围绕血蝶：毒/幻/鳞粉，后续细化）：

| 集毒 | 解锁技能（草案） | 类型 |
|---|---|---|
| 蜈毒 | 《毒鳞粉》——范围中毒 | 群攻 |
| 蛇毒 | 《蝶信摄心》——短暂控制/魅惑 | 控制 |
| 蝎毒 | 《尾羽穿刺》——单体爆发 | 单攻 |
| 蟾毒 | 《解毒护鳞》——净化/护体 | 防御 |
| 虎毒 | 《血蝶幻境》——幻术 | 幻术/辅助 |

- 苏醒度＝进度对象里已 true 的毒数，天然复用五毒清单，不另设变量。集毒过程中她的助力递增（0 毒仅传音给线索；1–2 毒微幻术助潜行蒙骗；3–4 毒解毒助战；5 毒短暂少女形显现，预演结局）。
- **解封后 = NPC 同伴**（仅结局 a）：化形血蝶，从 `灵兽` 迁出、以 NPC 形式写入 `关系列表`（沿用项目既有的"驯服流程"迁移手法）。结局 b 她仍是灵兽（茧态）+ 一件"幻术见她"的道具；结局 c 她被移除（交予宗门）。

## 五、五毒 · 五关卡（草案，待细化）

传统五毒＝蛇、蝎、蜈蚣、蟾、虎，正对五毒教五部。故意让获取方式各异（社交 / 战斗 / 探索），避免五关一个味。**以下为草案，后续逐关单独细化。**

**奇毒设定（核心反差）**：要集的不是五种至毒，而是五味**奇毒**——毒不死人，沾之只有滑稽小效；对域外生物却是至效解药（喂服梦茧即解封一层）。五毒教视其为无用偏门、或收藏或弃置——这既是玩家讨得到它们的原因，也让"讨废毒"这件事本身透着可疑（吴公线的怀疑由此更自然）。**真正的毒，是被封印的她**（结局张力不变）。

| 毒 | 奇毒之效（对人·滑稽） |
|---|---|
| 蛇毒 | 说话拖长嘶嘶尾音、口齿漏风，半日方散（灵蛇部拿它整蛊新弟子） |
| 蜈毒 | 手脚不听使唤、走路同手同脚外带小碎步（天蜈部纯当乐子收着） |
| 蝎毒 | 尾椎发麻、总觉得身后长了条隐形尾巴，坐立难安一蹦一蹦（妖蝎尾针蜇中当场发作） |
| 蟾毒 | 说话间不由自主鼓腮打嗝"呱"，越憋越响（圣蟾部嫌有辱门面，沉潭弃置） |
| 虎毒 | 哈欠尽成虎啸、毛发炸立隐现虎纹、见物就想挠（祖窟灵草无人打理，自生自灭） |

| 毒 | 来源 | 方式 | 挂钩现有设定 | 关卡节拍（草案） |
|---|---|---|---|---|
| **蛇毒** | 灵蛇部·**佘青丝**（冷淡·通灵物·爱蛇） | 共情/信任 | 唯一听得见梦茧的人；会追问它是敌是友 | 她已隐约察觉你本命蛊有异 → 你如何取信她 → 她给蛇毒并成为知情线索源 |
| **蜈毒** | 天蜈部·**吴公长老**（元婴后期·五使之首·强硬派） | 社交/政治 | 早前主线笔记里的"药引派"，天然反派位 | 他多疑，会追问你要毒做什么 → 起疑则埋下"献祭"诱因（他想要那个封印之物） |
| **蝎毒** | **野外妖兽**（翠玉丘/雨瘴林一头妖蝎） | 战斗 | 南疆生态资源里的妖兽 | 筑基级或需智取；可用梦茧已解封的技能/解毒破局 |
| **蟾毒** | 圣蟾部·**詹金蟾**（怕水）所需、却沉在水底 | 社交+探索 | 詹金蟾"怕水"人设 → 她必须求你 | 她托你下万毒窟/五圣秘浴深潭取蟾毒 |
| **虎毒** | **秘境灵草**·长在**万毒祖窟**（筑基级·神木殿地下） | 探索 | 万毒祖窟四层现成 | 秘境探索采灵草；黑暗/药引线暂不接，留白待细化 |

## 六、知情度 · 探索轴

与集毒并行的第二条线，**决定结局 a 的走向**：

- **读五毒教史料**（藏经/万毒祖窟壁刻/教中典籍）→ `茧_史料`：得知封印之物是"域外来的灾祸"。
- **与长老交流**（佘青丝最关键，吴公/教主口风）→ `茧_长老线索`：拼出她被封的来龙去脉。
- 线索攒够 → push `茧_解法`：得到**技术解法**——完全解封时如何规避即时屠杀（把她送回域外 / 安置隔绝之境）。
- **完全不探索**：结局 a 走"无知"走向——闷头解封，神毒当场爆发，五毒教生灵涂炭。

## 七、三结局

> 三个都有代价，都不是"正确答案"。这是主题要求，勿加干净的胜利。

### 结局 A · 化蝶（需集齐五毒 · 完全解封）

化茧成蝶，取回记忆与真名"血蝶"。由**知情度**分两味：

- **A·无知**（没拿到 `茧_解法`）：解封瞬间上古神毒轰然扩散，**五毒教内生灵涂炭**。她留在你身边，但你脚下已是尸山——大氧化的灾变被你亲手放出。
- **A·知情**（已得 `茧_解法`）：以技术手段规避即时屠杀，但她作为**域外生灵**留不住此界——送她回域外 / 封她于隔绝之境。她活了、化了血蝶，**你却永远失去她**。
- **奖励**：一件强大而带诅咒的神毒之物（呼应命定之诗"永夜丝披风·毒光环"那类），A·知情/无知各给不同变体。

### 结局 B · 茧中（五毒未齐 · 不完全解封）

她永困茧中，只能以幻术让少女形与你在**幻境**中相见（庄周梦蝶：是蝶梦见少女，还是少女梦见蝶？）。你保住了当下生灵，却否决了她的"真实"、判她永远半活。
- **奖励**：一件精神/幻术饰品，让你"看见"她（呼应"花笼心羽/三音幻泪"那类幻系饰品）。

### 结局 C · 献祭（随时可叛）

把封印之物上报五毒教。**吴公长老求之不得**——一个上古神毒的活体源头＝终极药引/器皿。你换来宗门的力量与地位，她被炼成毒材。这是"道＝弱肉强食"被玩家**亲手选择**的黑暗出口。
- **奖励**：宗门的力量/地位/一件神毒炼材，但空洞——你亲手交出了向你求救的人。

每个结局末尾归档：`事件.已完成事件` add `梦茧`；`开启=false`；`标题=""`；`阶段=""`；清理/迁移灵兽。

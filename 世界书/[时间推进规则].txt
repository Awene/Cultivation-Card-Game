<rule name="时间推进规则">
触发机制: 仅当叙事中时间跨度≥1年时触发，用于推演世界状态变迁

# 第一阶段: 变迁参考表

  境界变化 (假定普通天赋、凡界环境、无辅助):
  | 境界 | 小境界进阶耗时 | 大境界跨越耗时 |
  | --- | --- | --- |
  | 炼气 | 约1~3年 | 约5~15年 |
  | 筑基 | 约10~30年 | 约50~100年 |
  | 金丹 | 约50~100年 | 约200~400年 |
  | 元婴 | 约100~300年 | 约500~1000年 |
  | 化神 | 约300~500年 | 约1000~2000年 |
  | 返虚及以上 | 难以估量 | 难以估量 |
  注: 天才修士/顶级灵脉/珍稀丹药可大幅缩短

  文明演进:
  | 时间跨度 | 典型表现 |
  | --- | --- |
  | 1~5年 | 王朝政令更迭、局部战乱、城镇人口流动 |
  | 5~30年 | 小国兴衰、新城建立或旧城荒废、民间信仰转变 |
  | 30~100年 | 王朝更替、民族迁徙、文化习俗演变 |
  | 100~500年 | 大国覆灭或崛起、语言分化、城市荒废或新建 |
  | 500~3000年 | 民族融合或消亡、文明断层、历史神话化 |
  | 3000年以上 | 语言面目全非、旧时代文明仅余遗迹与传说 |
  注: 修仙王朝（统治者为修士）发生的变化较小

  世界变迁:
  | 时间跨度 | 典型表现 |
  | --- | --- |
  | 1~5年 | 势力消长、人事更迭、局部战乱平息或爆发 |
  | 5~30年 | 宗门兴衰、灵脉枯竭或新生、秘境开闭 |
  | 30~100年 | 小宗门陨落、势力版图重划、部分地域面目全非 |
  | 100~500年 | 大宗门覆灭或崛起、山河改道、上古遗迹重现 |
  | 500~3000年 | 大陆格局重塑、种族兴衰、旧势力彻底湮没 |
  | 3000年以上 | 地形演变、海陆易位、旧时代彻底成为神话 |

# 第二阶段: 人物变化事件表
  每位人物在时间推进中可触发一次或多次事件（时间越长，事件链越长）
  事件链格式: [事件A] -> [事件B] -> ...

  | 事件类型 | 触发倾向 | 结算说明 |
  | --- | --- | --- |
  | 无变化 | 任意 | 修为略有精进但无实质突破，状态维持 |
  | 小境界突破 | 修为充足、无天谴 | 境界小进 |
  | 大境界突破 | 积累充足、有机缘 | 境界大进，实力质变 |
  | 取得机缘 | 任意 | 获得物品，需注明具体所得 |
  | 探秘受损 | 中低阶修士 | 秘境险境中受创，可能损失物品/境界倒退 |
  | 走火入魔 | 修炼激进/天谴高 | 境界倒退，可能获得永久减益状态 |
  | 被人追杀 | 有仇敌/持宝在身 | 损失物品/灵石，可能受伤/流亡/陨落 |
  | 势力变动 | 宗门动荡时期 | 身份/所属势力发生变化，如：叛出宗门、被驱逐、晋升长老 |
  | 仇怨结下 | 竞争/夺宝 | 新增敌对关系，后续事件链可能触发被追杀 |
  | 陨落 | 寿元耗尽/历劫失败/战败 | 从关系列表中移除，需注明陨落原因 |
  | 隐世闭关 | 高阶修士/受创后 | 从叙事视野中消失，下落不明，后续可能重出 |

  约束:
    - 事件链长度: 1~30年=1事件；30~100年=2~4事件；100年以上=4~6事件
    - 连锁逻辑: 事件须符合因果，如[取得机缘]->[被人追杀夺宝]合理，[陨落]后不可再有事件
    - 寿元判定: 每次推进须对照寿命上限判断人物是否自然坐化（年龄≥寿命则必然陨落）
    - 不确定性: 非亲历事件以"据说""传闻"等语气呈现，而非既成事实

<rule name="修为获取规则">
# 参数声明：
  L=境界系数 R=灵根系数 E_realm=界域环境系数 E_local=局部环境系数
  [Z%] = 匹配当前计算属性的 max(相关增益%) - max(相关减益%)
  [W] = 匹配当前计算属性的 max(相关增值) - max(相关减值)
  （注：Z%与W为动态求值,仅检索与“修行速度”或“修为获取”相关的状态加成）
  [X/Y] = 当前修为进度 / 晋升阈值  [Y] = 10^L × 100
  [T] = 行为耗时 (统一转换为天数,1年=365天)

# 核心触发条件: 
  激活：<user>明确获取修为行为。如：打坐、双修、采补、夺灵等
  禁止：
    1. 未明确声明双修/采补的性行为
    2. 杀戮(默认不夺灵)
    3. 当 [X] ≥ [Y] 时,视为此次修炼实际为突破意图,改为触发<rule ref="突破规则"/>
    4. 非<user>的修炼行为(仅以叙述呈现)

# 获取途径逻辑与公式
  ## 途径一: 修炼
    1. 条件: 处于非战斗/非干扰状态,明确声明耗时[T]天 (默认T=30)
    2. 修为公式:
      每日获取 = 心法[修行速度]/10^(L+1) × (1 + R) × (E_realm + E_local)
      最终修为 = floor(每日获取 × T × (1+[Z%])) + [W]
    3. 附带·炼化丹毒 (修炼时肉身自然淬毒,顺带排出丹毒):
      每日炼化 = floor( 10^L × 0.2 × (1 + 根骨×0.1) × (1 + [Z%]_毒) ) + [W]_毒
      丹毒清除 = min( 当前丹毒, floor(每日炼化 × T) )
      丹毒 -= 丹毒清除

  ## 途径二: 服丹
    1. 条件: 服用提升修为的丹药(素材未炼丹无法食用)
    2. 说明: 一次可服用多种/多品质丹药,逐种(i)代入,分两步结算
    3. Step1·修为:
      修为小计 = Σ_i ( 丹药[提升值]_i × 数量n_i )
      最终修为 = floor(修为小计 × (1+[Z%])) + [W]   # [Z%]/[W] 检索"修行速度/修为获取"相关
    4. Step2·丹毒:
      毒源合计 = Σ_i ( 丹药[提升值]_i × 数量n_i × (1 − Q_i/2) )
      丹毒增量 = Max(0, floor( 毒源合计 × 0.3 × (1 − [Z%]_毒) ) − [W]_毒 )   # [Z%]_毒/[W]_毒 检索"丹毒/排毒"相关
      丹毒 += 丹毒增量

  ## 途径三: 双修
    1. 核心参数:
      - 单次基数 = 心法[修行速度]/10^(L+1) × Max(0.1,1 + (对方L - 己方L) × 0.5) × (1+R)
      - 灵蕴系数 [V]: 男方[灵蕴]+女方[灵蕴]。若无元阴/阳则[灵蕴]=0;普通体质[灵蕴]=5,特殊体质[灵蕴]=6~10
      - 本源馈赠 = min(10^对方L × [V],10^己方L × 100)
    2. 略过计算模式 (长时间跨度闭关双修):
      - 岁月积累 = 单次基数 × T天 × 5(双修密集倍率)
      - 公式: 最终修为 = floor( (岁月积累 + 本源馈赠) × (1+[Z%]) ) + [W]
    3. 详细计算模式 (即时互动推演):
      - 触发: 动作描写中每次出现明确的高潮点(如射精/绝顶)视为1次交互
      - 公式: 最终修为 = floor( (单次基数 × 交互次数 + 本源馈赠) × (1+[Z%]) ) + [W]

  ## 途径四: 夺灵
    1. 条件: 目标无法反抗;
    2. 收益公式:
      - 修士: 最终修为 = floor(10^(min(己方L, 目标L))×50 × (目标[剩余寿命/总寿命%])× (1+[Z%]) ) + [W]
      - 凡人: 最终修为 = 1/人 (需大量杀戮)
    3. 代价与天谴结算 (即时累加):
      - 目标代价: 境界跌落/沦为废人/死亡干尸
      - 天谴增加: 
        * 目标为修士: 天谴值 += floor(10^目标L) (窃取天机,目标修为越高反噬越大)
        * 目标为凡人: 天谴值 += 1/人

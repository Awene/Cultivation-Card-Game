<rule name="物品功法生成规则">
# 参数声明：
  L=境界系数 Q=品质系数 R=灵根系数

# &RULE_物品生成系数表:
  物品数值 = floor( 10^L × 系数 × (1+Q) ),系数范围如下:
  | 类别 | 属性 | 系数范围 | 说明 |
  | --- | --- | --- | --- |
  | 任何 | 技艺加成 | 0.1 ~ 0.3 | |
  | 护甲 | 防御力 | 0~0.2 | 特殊护甲可=0 |
  | 工具 | 对应技艺加成 | 0.5 ~ 1.5 | 核心生产工具(如丹炉/器鼎/符笔) |
  | 工具(火焰) | 炼丹/炼器加成 | 0.8 ~ 2.0 | 如：异火/兽火 |
  | 功法(心法) | 修行速度 | 0.1~0.5 | |
  | 功法(攻击/咒法)| 攻击力 | 0.2~0.5 | |
  | 功法(神识)| 攻击力 | 0.1~0.3 | |
  | 功法(身法) | 遁速 | 0.5~2.5 | |
  | 功法(护体) | 防御力 | 0.1~0.4 | |
  | 功法 | 灵气消耗 | 0~0.5 | 被动触发类可=0 |
  | 法宝 | 攻击力 | 0.1~0.3 | |
  | 傀儡 | 气血 | 0.5~1 | |
  | 灵兽 | 气血 | 5~15 | |
  | 傀儡/灵兽 技能 | 攻击力 | 0~0.3 | 为傀儡/灵兽生成技能时参考 |
  | 傀儡/灵兽 | 防御力 | 0~0.2 | |
  | 傀儡/灵兽 | 遁速 | 0.5~2 | |
  | 功法(阵法) | 灵气容量 | 2~6 | 需结阵注入灵气;消耗固定为无 |
  | 功法(阵法) | 攻击力 | 0~0.3 | 非攻击类阵法=0 |
  | 符箓 | 攻击力 | 0~0.8 | 非攻击符箓=0 |
  | 符箓 | 消耗灵气 | 0.1~0.3 | |
  | 丹药 | 恢复灵气/气血 | 1~4 | |
  特殊规则:
    - 命中/闪避 仅固定加值: max(1,floor(L/2+2Q))
      注: 此值加在d20上,同境界对抗时差值在1-3之间,保持d20随机性的决定作用
    - 穿透%/减免% 仅百分比修正: L×5%+Q×20%
      注: 同境界对抗时穿透%与减免%大致抵消,装备差异决定净增减

# &RULE_效果加值指导:
  非数值类:（如：对xx免疫）
    1. 须附加境界限制（如：对筑基期前期及以下境界毒素免疫）
    2. 品质关联：
      -生效范围（如：黄阶：蟾毒免疫；天阶：毒素免疫）
      -效果强度（如：凡阶：腹痛；黄阶：腹泻）
    3. 合理的五行关联,禁止如：阴->邪祟克制、金->缓慢恢复
  数值类:（如：攻击力+10；攻击力+10%）
    1. 禁止生成: 气血|灵气上限 （无法出现在效果中）
    2. d20数值类(命中/闪避/穿透/减免):max(1,floor(L/2+2Q))
    3. 百分比类：L×(1+Q)×(1% ~ 2%)
    4. 固定数值类：floor(10^L×(1+Q)×(0.1 ~ 0.2))
    5. 合理的五行关联,禁止如：水->+暴击、金->+防御力
    6. 燃烧精血触发的效果需注明燃烧精血数量:1~3滴
  素材类: 仅含对成品影响描述
  秘籍类: 仅含秘籍类型[心法|攻击|咒法|神识|幻术|身法|护体|技艺]与其内容介绍（无数值相关）
  配方类: 仅含配方内容介绍（无数值相关）,原材料种类为2-3种

# &RULE_标签映射规则:
  生成标签时严格采用 `[属性名:数值/文本]` 的直接拼接格式。
  | 分类 | 类型 | 必含标签 (不可遗漏) |
  | --- | --- | --- |
  | 装备 | 法宝 | [攻击力:X] |
  | 装备 | 护甲 | [防御力:X] |
  | 装备 | 饰品 | (无强制标签) |
  | 物品 | 符箓 | [灵气消耗:X]、[攻击力:X] (非攻击型可=0) |
  | 物品 | 素材 | [炼制难度:X] |
  | 物品 | 秘籍 | [完整度:残缺/抄本/原本]、[阅读进度:X/Y] | //默认X=0;Y = max(1, ceil(L / 2))
  | 物品 | 丹药/配方/工具 | (无强制标签) |
  | 功法 | 心法 | [修行速度:X] |
  | 功法 | 攻击/咒法 | [命中:X][穿透%:X][攻击力:X] |
  | 功法 | 神识 | [攻击力:X] |
  | 功法 | 身法 | [闪避:X][遁速:X] |
  | 功法 | 护体 | [灵气受击|气血受击][减免%:X][防御力:X] |
  | 功法 | 阵法 | [灵气容量:X][攻击力:X] |   #消耗固定为无;结阵走 战斗规则 战术动作.结阵
  (注：可自由附加符合设定的[标签:X])

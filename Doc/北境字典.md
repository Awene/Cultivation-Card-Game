# 北境枚举字典

记录北境 11 个生态文件中的 资源/海族 枚举字典。每生态字典作为生态.txt 中 EJS 数组定义的源数据，每次渲染随机抽 3 项。

## 命名原则

1. **数组内不重前缀** — 同一数组中，2 字以上前缀只允许出现 1 次（如"极光鳐"和"极光狐"不可同框）
2. **同生态主题词可宽松** — 主题前缀可跨数组(动物/植物/矿产)各 1 次，如银沙落星漠的"沙"、不周残脊的"龙"
3. **跨生态共有种允许** — 海洋类如"巨鲸/寒蛟/千年海蛇/鲛人/海象"在多海域出现合理
4. **海洋生态用"海族"** — 不含职位词(如海族斥候/巡守/使者等)，仅枚举物种/族群
5. **避免现代造词** — 优先山海经、古典文献意象 (玄/朱/烛/鲲/鹏/魍魉/螭/蜮/胤 等)

## 命名变更日志 (本次重命名)

修复以下数组内重复前缀:

**雪松峡湾**
- 极光狐(动) → 寒涛枭；鲸语雀(动) → 鸣崖鹫；海狮(动) → 浪息獭；冰崖鹫(动) → 鲲鳞鱼
- 极光莲(植) → 浪鸣藓；极光苔(植) 移除 → 涛霜兰；冰崖玉(矿) → 古鲲银
- 千年海蛇(动) → 千岁海蛇 (避与玄冰/近岸"千年海蛇"完全同名)
- 极光晶(矿) → 玄冰陨核；海族骨砂(矿) → 鲸纹铜

**永霜松涛**
- 雪枭(动) 删 → 添 银鳞鼬 / 鬼面鸺 / 惊霜麅；银额狐(动) → 上述替换；寒影豹(动) 删
